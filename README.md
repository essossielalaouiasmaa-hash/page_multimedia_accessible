# Accessible Multimedia Page

Ce projet est une page web contenant différents médias avec des fonctionnalités d'accessibilité.

## Objectif
Créer une page multimédia respectant les bonnes pratiques d'accessibilité.

## Contenu
- Vidéo avec contrôles natifs
- Sous-titres avec fichier .vtt
- Podcast audio
- Transcription textuelle
- Galerie d'images avec lazy loading
- Vidéo YouTube intégrée

## Accessibilité
- Texte alternatif pour les images
- Navigation clavier
- Sous-titres vidéo
- Transcriptions

## Technologies utilisées
- HTML5
- CSS
